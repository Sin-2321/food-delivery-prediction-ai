# 🍔 SmartBites AI: Predictive Food Delivery Delay Engine

SmartBites AI is a full-stack predictive machine learning web application built to accurately estimate food delivery latency. By analyzing factors like order volume, distance, weather severity, and traffic density, the engine computes real-time time estimates and assesses delivery "hunger risk" levels to help optimize logistics.

## 🚀 Live Demo
🔗 **[Click here to try the Live App!](https://food-delivery-prediction-ai-xx8seimjyammmouevm6gxm.streamlit.app/)**

---

## 📊 Key Features
* **Interactive Dashboard:** Dynamic user sliders for distance, item counts, weather status, and traffic density.
* **Predictive Core:** Uses a trained Machine Learning model to calculate accurate delivery timelines dynamically.
* **Risk Stratification:** Classifies output predictions into automated color-coded safety indicators (Low, Medium, High Risk).
* **Cloud-Deployed:** Hosted live on Streamlit Community Cloud for global accessibility.

---

## 🛠️ Technology Stack
* **Language:** Python 3
* **Machine Learning:** Scikit-Learn (Random Forest Regressor)
* **Data Processing:** NumPy, Pandas
* **Web Interface:** Streamlit
